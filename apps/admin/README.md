# FastAIBooking Admin Frontend

Platform admin application for:

- admin authentication
- global dashboard metrics
- salon lifecycle management
- cross-salon operations (staff, services, business hours, appointments, customers, billing usage)
- call logs and AI logs visibility

## Local Development

```bash
npm --workspace @fastaibooking/admin run dev
```

## Build

```bash
npm --workspace @fastaibooking/admin run build
```
