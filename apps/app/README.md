# FastAIBooking Shared App Frontend

Shared application for:

- salon owners (full business management)
- staff users (restricted operational workflows)

Role-based routing and navigation are enforced in the frontend, with matching backend authorization checks.

## Local Development

```bash
npm --workspace @fastaibooking/app run dev
```

## Build

```bash
npm --workspace @fastaibooking/app run build
```
